Adds an area graph to an nf-graph component.

Optionally, if it's located within an nf-area-stack component, it will work with sibling nf-area components to create a stacked graph.

trackingMode

String

Inherited from mixins.graph-graphic-with-tracking-dot: app-addon/mixins/graph-graphic-with-tracking-dot.js:13

Gets or sets the tracking mode of the component.

Possible values are:

  • 'none': no tracking behavior
  • 'hover': only track while mouse hover
  • 'snap-last': track while mouse hover, but snap to the last data element when not hovering
  • 'snap-first': track while mouse hover, but snap to the first data element when not hovering
  • 'selected-hover': The same as 'hover' tracking mode, but only when the compononent is mixins.graph-selectable-graphic/selected:property
  • 'selected-snap-last': The same as 'snap-last' tracking mode, but only when the compononent is mixins.graph-selectable-graphic/selected:property
  • 'selected-snap-first': The same as 'snap-first' tracking mode, but only when the compononent is mixins.graph-selectable-graphic/selected:property

Default: 'none'
