align

String

Defined in app/components/nf-column.js:55

Sets the alignment of the items in the column. Using this attribute adds appropriate styling for sort arrows.

To make sort arrows appear for center and left alignment, the text you want the sort arrow to appear next to needs to be wrapped in <span class="nf-column-label"></span>

This property is completely optional, and is here as a convenience. You don't need to use this property at all, if you which to add your own sort styling.

Default: null

sortDirection

String

Defined in app/components/nf-column.js:107

Gets or sets the sort direction for the column.

Possible Values:

  • 'asc' - sort ascending.
  • 'desc' - sort descending.
  • 'none' - (or any other value) no sort.

Default: 'none'

Example:

 {{#nf-column sortDirection="asc"}}
